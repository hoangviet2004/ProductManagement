Trước khi sửa code thì luôn làm bước này
git checkout master → git pull → git checkout 'nhánh của mình' → git merge master

Khi làm xong và muốn push code lên
git checkout master → git pull → git checkout 'nhánh của mình' → git merge master → git add . → git commit -m "nội dung" → git push origin 'nhánh của mình'
